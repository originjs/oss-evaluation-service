# oss-evaluation-service

## Getting Started
  create an `.env` file 
  ```
  DATABASE_URL='mysql://your_db_user:your_password@your_db_server/your_dev_db_name'
  ```
- Run
```bash
pnpm install
pnpm dev
```
- Swagger API
  
  http://localhost:3000/api-docs
